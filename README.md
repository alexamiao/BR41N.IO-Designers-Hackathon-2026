# g.tec 2026 Hackathon — BCI Visual Systems

This repository contains three interactive projects developed during the **g.tec Neurotechnology Spring School** and **BR41N.IO Designers’ Hackathon 2026**.

These projects explore how brain–computer interfaces (BCI) can be used to visualize and interpret human cognitive and sensory experiences.

---

## 🌸 NeuroBloom — Neuro-Responsive Sanctuary

NeuroBloom is a real-time biofeedback system that transforms brain activity into a growing digital garden.

By mapping EEG-derived states such as arousal, valence, and focus to visual elements, the system enables users to observe and interact with their internal mental states.

**Pipeline:**  
BCI Input → Signal Processing → State Classification → Visual Engine  

**Outcome:**  
An evolving, interactive garden that reflects the user’s mental state and supports self-awareness.

---

## 🔥 NeuroPalate — Brain Response to Taste

NeuroPalate visualizes how the brain responds to taste stimuli using EEG signals.

The project explores conditions such as **dysgeusia**, where taste perception is distorted, and investigates whether similar stimuli produce consistent neural responses across individuals.

**Pipeline:**  
Taste Stimulus → EEG Capture → Signal Processing → State Mapping → Visual Rendering  

**Experiment:**  
- 3 participants  
- 4 taste stimuli (salt, pepper, vinegar, sugar)  
- Machine learning model used for pattern analysis  

---

## 🌌 Awe of Scale — Experiential EEG Environments

This project studies neural responses to immersive environments of different scales.

Participants experienced:
- **Macro environments** (e.g., landscapes, aerial views)
- **Micro environments** (e.g., cellular-level visuals)

EEG signals were classified into:
**calm, relaxed, curious, engaged, and awe**

This work helps VR creators and studios design more engaging, attention-sustaining, and awe-inspiring experiences.
